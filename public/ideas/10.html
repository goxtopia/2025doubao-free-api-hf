Behavioral Anomaly Detector

Technique: Unsupervised Learning (Autoencoders) / LLM Contextualization

1. User Pain Point

Rule-based alerts (motion detect) are dumb. They miss subtle things like 'Front door open at 3 AM' if no motion occurs.

2. Abstract Need

Learning 'Normalcy' to detect 'Abnormality'.

3. Initial Solution

AI learns household patterns (lights off by 11 PM, dog sleeps in kitchen). Flags deviations.

4. Self-Correction / Refutation

Life is chaotic. Too many false alarms (party, late movie). Model drift.

5. Refined Strategy

Use LLM to explain the anomaly and ask for feedback. 'Usually, the garage is closed by now. It is open. Is this okay?' Learn from user responses.

6. Final Product Feature

The 'Gentle Nudge': A proactive notification system that questions unusual states rather than blaring alarms.
