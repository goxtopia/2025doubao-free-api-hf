Trash Bin State Monitor

Technique: Object Detection / Classification

1. User Pain Point

Forgetting to take the trash out on Tuesday night; Missing the truck.

2. Abstract Need

Routine chore compliance.

3. Initial Solution

Camera checks driveway. If 'Bin' is not visible on Curb on Tuesday 9 PM -> Alert.

4. Self-Correction / Refutation

Simple but effective. Challenge: Differentiating 'Bin at side of house' vs 'Bin at curb'.

5. Refined Strategy

User sets 'Curb Zone'. If Bin is not detected in 'Curb Zone' within time window, send reminder.

6. Final Product Feature

Curbside Reminder: A location-aware alert that nags you only if you actually forgot the chore, preventing smell accumulation.
