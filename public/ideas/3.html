Active Porch Pirate Deterrent

Technique: Action Recognition (I3D/SlowFast) + Object Detection

1. User Pain Point

Package theft happens in seconds; recording it just provides a movie of your loss.

2. Abstract Need

Real-time intervention during a specific adverse event.

3. Initial Solution

Detect a person approaching a delivered package -> Grab -> Run. Trigger siren/strobe immediately.

4. Self-Correction / Refutation

False positives (delivery driver moving the package, owner picking it up) are annoying and embarrassing.

5. Refined Strategy

Use face recognition (Owner whitelist). Detect specific 'grab and dash' mechanics vs 'pickup and walk'. Warning voice first: 'Hi, you are being recorded' before siren.

6. Final Product Feature

Context-aware Deterrence System that escalates from polite greeting to verbal warning to alarm based on interaction with the package.
