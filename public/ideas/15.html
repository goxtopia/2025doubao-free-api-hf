Garage Precision Parking

Technique: Keypoint Detection / Optical Flow

1. User Pain Point

Parking in tight garages is stressful. Tennis ball on a string is primitive.

2. Abstract Need

Assisted vehicle positioning.

3. Initial Solution

Use the garage security camera to track the car's bumper. Flash smart lights red when to stop.

4. Self-Correction / Refutation

Latency again. If the light turns red 500ms late, you hit the wall.

5. Refined Strategy

Predict velocity to flash early. Provide progressive feedback (Green -> Yellow -> Red) via smart bulb brightness or color.

6. Final Product Feature

Visual Parking Sensor: Uses existing garage cam to turn smart lights into a traffic signal, guiding perfect parking without new hardware.
