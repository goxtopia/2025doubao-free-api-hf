Appliance State Verification

Technique: VLM (Visual Language Model) / YOLO

1. User Pain Point

Anxiety about leaving the stove, iron, or heater on after leaving the house.

2. Abstract Need

Remote state verification of physical objects.

3. Initial Solution

User asks 'Is the stove on?'. VLM analyzes the latest frame of the kitchen camera for knobs/flame indicators.

4. Self-Correction / Refutation

Knobs are small and low-res. Occlusion is common. Hallucination risk is high ('It looks off').

5. Refined Strategy

Combine with thermal imaging overlay if available, or use change detection (compare 'now' vs 'known off state'). Output confidence scores. Ask user to verify with a zoomed crop.

6. Final Product Feature

On-demand 'Appliance Check' that provides a zoomed-in snapshot and a VLM-based probability assessment, flagging high-heat signatures.
